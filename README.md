# Targ-Masini-PIU-2026

Inregistrarea detaliilor referitoare la masini care au fost vandute / cumparate pentru un targ de masini.

ㅤ
ㅤ

Pana la finalizarea proiectului, vor exista urmatoarele functii:

  -Introducere / Editare / Stergere date;
  
  -Avertisment la introducerea unei noi tranzactii daca o persoana cumpara / vinde mai multe masini in aceeasi zi;
  
  -Filtre de cautare:  
  
    -Data inregistrarii / tranzactiei / fabricarii;  
    -Pret;  
    -Model;  
    -Culoare;  
    -An fabricatie;  
    -Vanzator;  
    -Cumparator;  
    
  -Afisare rapoarte:
  
    -Cea mai cautata masina / firma / model intr-o perioada;
    -Grafic al pretului pentru un anumit model intr-o perioada;
    -Afisarea tranzactiilor dintr-o anumita zi;


ㅤ
ㅤ


Pe langa aceste functii, doresc sa mai adaug:

    -Notite legate de toate functiile implementate;
    -Informatii legate de accidente in care a fost implicata masina, dupa caz;
    -Data ultimei inspectii tehnice;
    -Notite legate de modificari semnificative asupra autoturismului;
    -Prezentarea kilometrilor totali parcursi de vehicul;

Pe parcursul realizarii acestui proiect noi functii ar putea fi adugate la lista de mai sus.
